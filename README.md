# Review Analysis Project 🔍📊

## Overview

A comprehensive sentiment analysis tool that leverages modern web technologies to analyze and visualize user reviews and sentiments.

![Project Banner](/path/to/project-banner.png)

## 🚀 Features

- **Real-time Sentiment Analysis**
- **Telegram Bot Integration**
- **Interactive Dashboard**
- **Multi-Platform Support**

## 📋 Prerequisites

- Node.js (v14+ recommended)
- npm or Yarn
- Telegram Bot Token

## 🔧 Installation

### Clone the Repository
```bash
git clone https://github.com/yourusername/review-analysis-project.git
cd review-analysis-project
```

### Install Dependencies
```bash
# Install backend dependencies
cd backend
npm install

# Install frontend dependencies
cd ../frontend
npm install
```

## 🔐 Configuration

Create a `.env` file in the backend directory:
```env
PORT=3001
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
```

## 🚀 Running the Application

### Development Mode
```bash
# Start Backend
cd backend
npm run start

# Start Frontend (in another terminal)
cd frontend
npm run start
```

### Production Build
```bash
# Build Frontend
cd frontend
npm run build

# Start Backend with production build
cd ../backend
npm run start:prod
```
